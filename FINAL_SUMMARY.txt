🔗 API ENDPOINTS

  POST /api/agency-search/save
    → Save agency + creators to database
    → Returns: { success, agencyId, creatorCount, creators[] }

  GET /api/agency-search/list
    → List all your agencies (RLS protected)
    → Returns: { success, count, agencies[] }

  DELETE /api/agency-search/:agencyId
    → Delete agency + all creators
    → Returns: { success, message }
