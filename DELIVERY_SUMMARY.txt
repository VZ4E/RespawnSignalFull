ARCHITECTURE HIGHLIGHTS
================================================================================

Design Pattern:
  • Modular: Services separated from routes
  • Reusable: Works with existing auth/credit system
  • Extensible: Same pattern for Instagram/Twitch next
  • Tested: Error scenarios covered
  • Documented: Every function commented

Key Decision: Try/Finally Credit Deduction
  • YouTube API fails? → No credit lost
  • Scan runs but AI fails? → Credit deducted (fair)
  • Scan completes? → All saved + credited
  • Crash mid-scan? → Finally still executes

Why 7-Day Cache vs 1-Day:
  • YouTube channels change less frequently than TikTok
  • Saves API quota for more creators
  • Users expect more stable descriptions
  • TikTok: rapid content, fast expiry
  • YouTube: lasting content, longer cache
