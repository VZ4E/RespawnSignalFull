### Key Features

**Scanner Capabilities**
- Resolves: @handle, channel ID, full URLs
- Depth: 10/20/30 videos per scan
- Speed: 5-10 seconds (vs TikTok's ~30 seconds)
- Detection: Brand names, deal types, promo codes, affiliate links
- Aggregation: Deduplicates brands, tracks appearances across videos

**Architecture Decisions**
- Uses Perplexity's online model (same as TikTok scanner)
- Temperature 0.1 (deterministic results)
- 7-day cache vs TikTok's 1-day (channels change slower)
- 1 credit per video analyzed (same as TikTok)
- Try/finally credit deduction (fair accounting)

**Error Handling**
- 404: Channel not found
- 400: Invalid input format
- 402: Out of credits
- 502: YouTube API unavailable
- 500: Unexpected failure
- All with user-friendly messages
